# 安装  

`batScript`的安装很简单.  

1. 从[GitHub Release]()下载最新的`batScript.zip`, 并解压到合适目录  
2. 将目录下的`env/`添加至环境变量  

完成后, 打开终端, 执行以下命令进行验证.  
编译器:  

```cmd
bsc -Version
```

包管理器:  

```cmd
bspm /v
```

## 附加程序安装  

`batScript`带有两个附加程序:  

- `BatScriptIDE`: 一个使用`WinUI3`编写的, 原生的`batScript`编辑, 运行整合软件.  
- `batscript-code-generator`: 一个使用`lua`编写, [luainstaller](https://github.com/Water-Run/luainstaller)打包的简单命令行工具, 用于生成和修改`.bs`脚本  

附加程序可以在[GitHub Release]()进行下载.
