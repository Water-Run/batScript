# 计划未来添加的功能  

以下是一些计划在`batScript`未来版本中添加的功能:  

## `b-string`  

`b-string`类似`python`中的`f-string`,用于格式化拼接字符串:  

```batscript
set b_string = "b-string";
set var_str = `this is a {b-string}`
```
