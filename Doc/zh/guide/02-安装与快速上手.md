# 安装和快速上手  

`教程` `2025-12-12`  

---

本教程指导如何下载, 安装`BatScript`, 和基础使用.  

## 下载和安装

顺序执行以下步骤, 即可完成安装:

1. 从[GitHub Release](https://github.com/Water-Run/BatScript/releases/tag/0.1.0)界面下载`BatScript-0.1.0.zip`  
2. 解压文件到你选择的目录  
3. 运行`install.bat`, 选择安装策略. 如果是初次安装, 建议使用`default`  
4. 重启电脑  

> 你的设备上可能需要额外下载`.NET`运行时环境.  

### 验证安装

按`Win`+`R`快捷键打开运行, 输入`wt`(Windows 11)/`powershell`/`cmd`(更老的设备)打开终端. 输入以下指令:

```powershell
BatScript Get-Version
```

如果出现以下输出, 说明安装成功:

```powershell
BatScript@WaterRun: Version 0.1.0
See more on https://github.com/Water-Run/BatScript :-)
```

## 编译`BatScript`  

将`.bs`编译为`.bat`非常简单. 指令如下:

```powershell
BatScript Compile-Code [待编译的.bs文件路径]
```

例如, 假设目录下有编写好的`D:\\Demo.bs`. 只需要在该目录打开终端, 输入并执行以下指令:

```powershell
BatScript Compile-Code Demo.bat
```

代码将编译到同目录下. 如果编译成功, 将显示:

```powershell
Successfully compiled to D:\\Demo.bat.
```

此时`.bs`已经编译为`.bat`文件, 你可以运行.  

> 你可以使用`-Output`设置输出路径. `BatScript`在编译时的行为可有更多参数控制, 参见[BatScript指令集说明]()  

## 编写`BatScript`  

`BatScript`自带了一个简单的`IDE`, 提供语法高亮, 提示, 一键编译与运行等基本功能. 如果你在刚刚的安装中使用缺省(`default`)行为, 你应该已经能在你的桌面快捷方式看到它了(名称叫`BatScriptIDE`).  

双击启动, 即可开始编辑.  

> 由于自带编辑器使用`WinUI3`编写, 你的系统至少需要`Windows 10 1809`  

你也可以使用`BatScript`支持的`Visual Studio Code`和`Visual Studio`插件, 获取语法支持: 在插件市场中搜索`batScript`即可.  

让我们从经典的`HelloWorld`开始我们的`BatScript`编程之旅:  

```batscript
echoln "HelloWorld"; # 打印HelloWorld
```

你可以尝试进行编译并运行生成的`.bat`脚本.
