|源类型|目标类型|逻辑说明|示例|
|---|---|---|---|
|`num`|`str`|转换为数字的字符串表示|`str@3.14 == "3.14"`|
|`num`|`bol`|`0`转换为`false`,其它转换为`true`|`bol@0 == false` `bol@1 == true`|
|`num`|`arr<num>`|一项的一维数组,唯一项的值为该数字|`arr<num>@10 == [10]`|
|`num`|`arr<bol>`|一项的一维数组,唯一项的值为该数字转换为`bol`后的值|`arr<bol>@-1 == [true]`|
|`num`|`arr<str>`|一项的一维数组,唯一项的值为该数字转换为`str`后的值|`arr<str>@0.1234 == ["0.1234"]`|
|`str`|`num`|第一个数字之前的`-`(负号)和`+`(正号),数字部分及第一个`.`转换拼接为数字,默认为0.|`num@"3.14" == 3.14` `num@"这-0是+一个00混1杂0-0.数字2的34字符串" == -100.234`|
|`str`|`bol`|空字符串`""`为`false`,其它为`true`|`bol@"123" == true` `bol@"" == false`|
|`str`|`arr<num>`|一项的一维数组,唯一项的值为该字符串转换为`num`的值|`arr<num>@"+数1字2" == [12]`|
|`str`|`arr<bol>`|一项的一维数组,唯一项的值为该字符串转化为`bol`的值|`arr<bol>@"false" == [true]`|
|`str`|`arr<str>`|一项的一维数组,唯一项的值为该字符串|`arr<str>@"This is a string array." == ["This is a string array."]`|
|`bol`|`str`|转换为该布尔值的字符串表示|`str@true == "true"` `str@false == "false"`|
|`bol`|`num`|`false`转换为`0`,`true`转换为`1`|`num@true == 1` `num@false == 0`|
|`bol`|`arr<num>`|一项的一维数组,唯一项的值为该布尔值转换为`num`的值|`arr<num>@true == [1]` `arr<num>@false == [0]`|
|`bol`|`arr<bol>`|一项的一维数组,唯一项的值为该布尔值|`arr<bol>@true == [true]` `arr<bol>@false == [false]`|
|`bol`|`arr<str>`|一项的一维数组,唯一项的值为该布尔值转换为`str`的值|`arr<str>@true == ["true"]` `arr<str>@false == ["false"]`|
|`arr<num>`|`num`|返回一维数组的长度|`num@[3.14, 2.71] == 2` `num@[] == 0`|
|`arr<num>`|`str`|将一维数组转换为字符串表示,保留`[]`|`str@[1, 2, 3] == "[1,2,3]"` `str@[] == "[]"`|
|`arr<num>`|`bol`|空一维数组为`false`,非空一维数组为`true`|`bol@[1, 2] == true` `bol@[] == false`|
|`arr<num>`|`arr<str>`|将一维数组中每个数字元素转换为字符串|`arr<str>@[1.5, 2.7] == ["1.5", "2.7"]`|
|`arr<num>`|`arr<bol>`|将一维数组中每个数字元素转换为布尔值|`arr<bol>@[0, 1, -5] == [false, true, true]`|
|`arr<str>`|`num`|返回一维数组的长度|`num@["hello", "world"] == 2` `num@[] == 0`|
|`arr<str>`|`str`|将一维数组转换为字符串表示,保留`[]`|`str@["hello", "world"] == "[hello,world]"` `str@[] == "[]"`|
|`arr<str>`|`bol`|空一维数组为`false`,非空一维数组为`true`|`bol@["a", "b"] == true` `bol@[] == false`|
|`arr<str>`|`arr<num>`|将一维数组中每个字符串元素转换为数字|`arr<num>@["1.5", "abc2.7"] == [1.5, 2.7]`|
|`arr<str>`|`arr<bol>`|将一维数组中每个字符串元素转换为布尔值|`arr<bol>@["hello", "", "world"] == [true, false, true]`|
|`arr<bol>`|`num`|返回一维数组的长度|`num@[true, false] == 2` `num@[] == 0`|
|`arr<bol>`|`str`|将一维数组转换为字符串表示,保留`[]`|`str@[true, false] == "[true,false]"` `str@[] == "[]"`|
|`arr<bol>`|`bol`|空一维数组为`false`,非空一维数组为`true`|`bol@[true, false] == true` `bol@[] == false`|
|`arr<bol>`|`arr<num>`|将一维数组中每个布尔值元素转换为数字|`arr<num>@[true, false, true] == [1, 0, 1]`|
|`arr<bol>`|`arr<str>`|将一维数组中每个布尔值元素转换为字符串|`arr<str>@[true, false] == ["true", "false"]`|
