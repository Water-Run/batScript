# 关键字表  

|名称|说明|
|---|---|
|`let`|声明一个常量|
|`set`|声明一个变量|
|`func`|声明一个函数|
|`proc`|声明一个过程|
|`return`|函数的返回值|
|`arr`|声明一个数组|
|`num`|数字类型|
|`str`|字符串类型|
|`bol`|布尔类型|
|`if`|条件判断|
|`elif`|条件分支|
|`else`|条件其它|
|`match`|条件匹配|
|`condtion`|条件匹配条件|
|`while`|基础循环|
|`loop`|无限循环|
|`repeat`|次数循环|
|`iterate`|可迭代循环|
|`circle`|可迭代无限循环|
|`break`|中断循环|
|`continue`|继续循环|
|`jumpout`|跳出至最外层控制流|
|`err`|抛出异常|
|`endprogram`|结束程序|
|`echo`|输出一行|
|`get`|接受输入|
|`pause`|暂停脚本|
|`runraw`|原生内嵌|
|`lib`|导入库|
|`pkg`|导入包|
|`invisible`|对外隐藏|
